# Electrochemical ML Analysis

This repository contains machine learning models to predict electrochemical current response of TiO2–MnO2-based electrode systems.

## Features
- Multiple ML models (RF, GB, SVR, KNN, MLP, etc.)
- Stacking Regressor (best performance)
- Scan rate-based analysis
- Automatic plotting and Excel export

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run script:
   python src/main.py

3. Upload Excel files when prompted (Google Colab)

## Output
- Scatter plots
- Model comparison graphs
- Excel results file
